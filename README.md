# NLW-1.0
Ecoleta - a marketplace to collect waste - Next Level Week from Rocketset

## Server project
### Set up
```bash
cd server
yarn install

yarn run db:migrate
```

### Run application
```bash
cd server && yarn dev
```

### Database commands
```bash
yarn run db # to run knex commands
yarn run db:up # to run knex migrate:up
yarn run db:down # to run knex migrate:down
```
